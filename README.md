# 🛒 E-Commerce with React

A modern E-Commerce web application built using **React + Vite** that fetches products from the DummyJSON API. Users can browse products by category, view detailed information, and manage their shopping cart using React Context API.

---

## 🚀 Features

- Browse products by category
- Hero image slider
- Product details page
- Related products section
- Shopping Cart using Context API
- Cart status saved with LocalStorage
- Loading Skeleton while fetching data
- Responsive Design
- Fast navigation with React Router

---

## 🛠 Technologies Used

- React.js
- Vite
- React Router DOM
- Context API
- LocalStorage
- DummyJSON API
- React Icons
- CSS3

## 📦 Installation

Clone the repository

```bash
git clone https://github.com/yourusername/E-commerce-with-React.git
```

Go to the project directory

```bash
cd E-commerce-with-React
```

Install dependencies

```bash
npm install
```

Run the project

```bash
npm run dev
```
